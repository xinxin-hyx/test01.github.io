# 物种展示项目 (Species Showcase)

## 项目简介

物种展示项目是一个现代化的响应式网页应用，用于展示不同物种的详细信息。项目采用简洁优雅的设计风格，提供流畅的用户体验和丰富的交互功能。

## 主要功能

- **响应式设计**: 完美适配桌面端、平板和移动设备
- **弹窗交互**: 点击卡片查看详细信息和生物列表
- **图片控制**: 支持图片放大、缩小和重置功能
- **表格功能**: 支持表格排序和内容筛选
- **动画效果**: 流畅的过渡动画和加载效果

## 技术特点

- **纯前端实现**: 无需后端服务器，可直接部署
- **模块化代码**: HTML、CSS、JavaScript分离，便于维护
- **性能优化**: 优化的代码结构和资源加载
- **跨浏览器兼容**: 支持现代主流浏览器

## 快速开始

### 本地运行

1. 解压项目文件
2. 使用任意HTTP服务器启动项目

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx http-server -p 8000
```

3. 在浏览器中访问 `http://localhost:8000`

详细的部署说明请参考 [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md)
